# 活动专题页目录

该目录用于统一存放 Zup 活动专题 H5 页面。

当前页面：
- `beijing-spring-awakening.html`（北京春醒交友季 - 外部 H5 分享页，含下载/打开 App 引导）
- `beijing-spring-awakening-inapp.html`（北京春醒交友季 - App 内嵌专题页，无下载/打开入口）
- `beijing-spring-awakening-venue.html`（商户专属据点页 - 模板，供商户转发朋友圈，只展示单个商户）

### 商户据点页使用说明

`beijing-spring-awakening-venue.html` 是单商户专属页面模板。为每个商户生成页面时，需替换文件中标有 `★ 商户定制` 注释的内容：

1. **页面标题 & SEO meta**：商户名称、描述、分享图片
2. **Hero 封面**：商户门店/环境封面图
3. **商户名 & tagline**：一句话描述商户特色
4. **地址、时间、标签**：据点实际信息
5. **关于我们**：商户自我介绍文案
6. **图集**：3 张店内环境照
7. **活动内容**：该商户春醒期间具体活动
8. **福利**：该商户提供的签到福利
9. **微信分享**：标题/描述/图片
10. **结尾商户名**

后续新活动页面请继续放在此目录下。
