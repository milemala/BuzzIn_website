# 活动抓取与审核服务

活动抓取、本地审核台、豆瓣脚本与 `review.db` 在仓库内独立目录维护，与 `events/` 下 H5 原型分开。

**目录：** [`../zup-event-crawl`](../zup-event-crawl)

**启动审核台：**

```bash
cd zup-event-crawl
npm start
```

浏览器打开 http://127.0.0.1:8787/

**完整交接文档（必读）：** [`../zup-event-crawl/docs/HANDOFF.md`](../zup-event-crawl/docs/HANDOFF.md)
